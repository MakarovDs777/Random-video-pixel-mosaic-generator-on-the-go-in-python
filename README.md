# Random-video-pixel-mosaic-generator-on-the-go-in-python

В первых 3 версиях разные пиксели перемешиваются между собой в кадре а в остальных 3 случайные мозаики этих пикселей то есть изображение разделяется на массив чисел по три числа RGB цвета, и мы последовательно берем массивы по порядку, и мы генерируем случайное число который будет разрезать этот массив с массивами RGB чисел на случайные доли массивов не больше количества RGB массивов в изображении, и дальше эти доли массивов с массивами ну скажем пускай там 3 доли от 1 до 100, 5 и 5 массивов по порядку заносятся в новый массив мозаику, и потом генерируется новое число скажем 29 и в него заносятся 29 массивов RGB эти случайные последовательности мозаиков этих массивов перемешиваются между собой, и соединяются в новый кадр, и так с разных картинок делаются ровно того же размера те же самые доли с первой выбранной картинки мозаики разных картинок объединяются в единый новый случайный кадр который при каждой последующей итерации случайно перемешивается до бесконечного цикла.
